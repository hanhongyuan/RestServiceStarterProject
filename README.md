# Rest Service Starter Project # 
Rest Api Starter project based on Spring boot (Security with JWT, Spring Security)

## How To Use ##
* Get Token,  POST :  http://127.0.0.1:8080/api/v1/auth with data : 

  ```json
  {
   "username" : "test",
   "password" : "test"
  }
  ```
  OR 
  
  ```json
  {
   "username" : "ridwan",
   "password" : "ridwan"
  }
  ```
